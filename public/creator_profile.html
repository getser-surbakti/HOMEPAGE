Books I've Read

Below are the books I've read along with summaries and notes for each. Click on the titles to explore detailed notes and takeaways.

"A Handbook of Agile Software Craftsmanship" by Robert C. Martin

A deep dive into agile practices and software craftsmanship. This book highlights key principles that help developers build maintainable software.

"The Pragmatic Programmer" by Andrew Hunt and David Thomas

A must-read for every software developer, filled with practical advice and best practices for achieving excellence in programming.

"SQL For Dummies" by Allen G. Taylor

An easy-to-understand guide for SQL beginners, covering everything from database design to complex queries.

"Automate the Boring Stuff with Python" by Al Sweigart

Automate the Boring Stuff with Python teaches simple programming skills to automate everyday computer tasks.

"Python for Data Analysis" by Wes McKinney

This book is comprehensive and will help you go from understanding the basics of data structures like Series and DataFrame to mastering the tools needed for real-world data analysis with Python.

"Data Science for Business" by Foster Provost & Tom Fawcett

This book provides a comprehensive introduction to data science concepts, especially focusing on their application to business problems. It's a great starting point for understanding how data analysis ties into decision-making processes in business.
