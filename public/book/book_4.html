Chapter 5: Dictionaries

What is a Dictionary?

  • Imagine you have a box where you can put things, but the cool part is that this box has special labels (called keys) on the outside. Each label helps you find the thing inside the box (called the value).
  • Example dictionary items:
    • "apple" ‚Üí "a red fruit"
    • "dog" ‚Üí "a fluffy animal"
    • "car" ‚Üí "a fast machine"
  • This is what we call a dictionary in Python!

1. Creating Dictionaries

  • To create a dictionary, write the keys and values inside curly braces {}.
    • my_dictionary = {
    "apple": "a red fruit",
    "dog": "a fluffy animal",
    "car": "a fast machine"
}
  • In this example, "apple", "dog", and "car" are the keys, and "a red fruit", "a fluffy animal", and "a fast machine" are the values.

2. Accessing and Modifying Dictionaries

  • You can access the value inside the dictionary using the key, like opening a box with a label!
    • print(my_dictionary["apple"])  # Output: a red fruit
  • To change the value of a key, just assign a new value to the key:
    • my_dictionary["apple"] = "a green fruit"
print(my_dictionary["apple"])  # Output: a green fruit

3. Iterating Over Dictionaries

  • To go through the entire dictionary using a loop:
    • If you want to iterate through the keys:
      • for key in my_dictionary:
    print(key)  # Prints 'apple', 'dog', 'car'
    • If you want to iterate through the values:
      • for value in my_dictionary.values():
    print(value)  # Prints 'a green fruit', 'a fluffy animal', 'a fast machine'
    • If you want to see both the key and the value (key-value pair):
      • for key, value in my_dictionary.items():
    print(key, ":", value)
    # Prints:
    # apple : a green fruit
    # dog : a fluffy animal
    # car : a fast machine

4. Common Dictionary Methods

  • get(): This method helps you find a value by key, and if the key doesn‚Äôt exist, it won‚Äôt cause an error but return a default message:
    • print(my_dictionary.get("apple"))  # Output: a green fruit
print(my_dictionary.get("banana", "No such fruit"))  # Output: No such fruit
  • keys(): This method returns all the keys in the dictionary:
    • print(my_dictionary.keys())  # Output: dict_keys(['apple', 'dog', 'car'])
  • values(): This method returns all the values in the dictionary:
    • print(my_dictionary.values())  # Output: dict_values(['a green fruit', 'a fluffy animal', 'a fast machine'])
  • items(): This method returns a list of all the key-value pairs:
    • print(my_dictionary.items())  # Output: dict_items([('apple', 'a green fruit'), ('dog', 'a fluffy animal'), ('car', 'a fast machine')])

In Summary:

  • Dictionaries are like boxes with labels (keys) and treasures (values).
  • You can look up treasures by keys, change them, and loop through all the keys or values.
  • Methods like get(), keys(), values(), and items() make it easy to work with dictionaries.

Chapter 6: Manipulating Strings

What are Strings?

  • A string is a piece of text! It can be a word, a sentence, or even a whole paragraph.
    • "Hello"
    • "I love Python!"
    • "This is a string!"
  • Now, let's learn some cool tricks for manipulating strings!

1. String Methods

  • String methods are like magic wands that change how your text behaves. Here are some common methods:
    • .upper(): Makes all the letters uppercase (big letters).
      • text = "hello"
print(text.upper())  # Output: "HELLO"
    • .lower(): Makes all the letters lowercase (small letters).
      • text = "HELLO"
print(text.lower())  # Output: "hello"
    • .split(): Splits a string into smaller parts based on a separator.
      • text = "I love Python"
words = text.split()  # Splits by spaces into words
print(words)  # Output: ['I', 'love', 'Python']
    • .join(): Joins a list of words back into a string.
      • words = ['I', 'love', 'Python']
sentence = " ".join(words)  # Joins the words with spaces
print(sentence)  # Output: "I love Python"
    • .replace(): Replaces part of a string with something else.
      • text = "I love Python"
new_text = text.replace("Python", "coding")
print(new_text)  # Output: "I love coding"
    • .strip(): Removes extra spaces from the beginning and end of the string.
      • text = "  Hello!  "
clean_text = text.strip()
print(clean_text)  # Output: "Hello!"

2. String Formatting

  • Sometimes, you want to create strings that change based on values. You can use f-strings or the format() method to fill in parts of the string.
    • Using f-strings: This is like having a template for your strings:
      • name = "John"
age = 10

    • Using format(): Another way to format strings with placeholders:
      • name = "John"
age = 10
message = "Hello, {}! You are {} years old.".format(name, age)
print(message)  # Output: "Hello, John! You are 10 years old."

3. Regular Expressions (Regex)

  • Regular expressions (Regex) help you find patterns in text, like searching for specific words, numbers, or characters.
    • Example: Searching for the word "Python":

    • Example: Finding all numbers in a string:
      • import re

    • Example: Replacing words with Regex:
      • import re

text = "I love Python!"

print(new_text)  # Output: "I love coding!"

In Summary:

  • String methods help you manipulate text in various ways, like turning it into uppercase, splitting it, or joining it back together.
  • String formatting helps create dynamic text with placeholders.
  • Regular expressions (Regex) are like powerful search tools for finding patterns in text.

Chapter 7: Pattern Matching with Regular Expressions

What‚Äôs a Regular Expression (Regex)?

  • Think of regular expressions (regex) like a super-powerful tool for searching through text. It's like using a magic magnifying glass that helps you find patterns in words, sentences, or anything made of letters and numbers.
  • For example, if you have a list of phone numbers, you can use regex to find all numbers that start with the area code "123".

Regex Syntax

  • "Syntax" just means the rules for how to write regex. It‚Äôs like a secret language with special rules. Here are some basic components:
    • . (Dot): Matches any letter or number (wildcard).
    • \d: Matches any digit (number). For example, \d can match 3, 7, 0, etc.
    • \w: Matches any letter, number, or underscore (a-z, 0-9, _).
    • \s: Matches a space, tab, or newline (used to match blank spaces between words).
    • ^: Means the start of a string. For example, ^Hello will only match "Hello" at the beginning of the text.

re Module in Python

  • In Python, the re module gives you all the tools to work with regular expressions. Think of it like a toolbox that lets you search, match, and manipulate strings using regex patterns.
    • import re  # We bring the toolbox into our code

# Use re.search to find if something exists in a text
result = re.search('cat', 'The cat is cute')
print(result)  # This will tell us that 'cat' is found in 'The cat is cute'

Common Regex Functions

  • re.search(): Searches for a pattern in a string. If it finds the pattern, it returns a match.
    • result = re.search('cat', 'The cat is cute')
  • re.match(): Checks if the pattern matches only at the beginning of the string.
    • result = re.match('cat', 'The cat is cute')  # Won't match, as 'cat' is not at the start.
  • re.findall(): Finds all occurrences of a pattern and returns them as a list.
    • result = re.findall('cat', 'The cat and the fat cat')
print(result)  # Output: ['cat', 'cat']
  • re.sub(): Replaces occurrences of a pattern with a new value.
    • result = re.sub('cat', 'dog', 'The cat is cute')
print(result)  # Output: 'The dog is cute'

Special Characters in Regex

  • \d: Matches any digit (number). For example, \d can match 1, 2, 5, etc.
  • \w: Matches any word character (letters, numbers, and underscores).
  • \s: Matches spaces, tabs, or newlines.
  • ^: Matches the start of a string.

Examples to Tie It All Together

  • re.search() example:
    • import re
text = 'I have 2 apples and 10 bananas.'
result = re.search(r'\d', text)  # Searching for any digit
print(result)  # Output: '2'
  • re.findall() example:
    • text = 'I have 2 apples and 10 bananas.'
result = re.findall(r'\d', text)  # Finding all digits
print(result)  # Output: ['2', '1', '0']
  • re.sub() example:
    • text = 'I love cats, cats are cute.'
result = re.sub(r'cats', 'dogs', text)  # Replaces 'cats' with 'dogs'
print(result)  # Output: 'I love dogs, dogs are cute.'

Conclusion

  • Regex is like a secret language for searching and changing text. You can use it to find patterns, check for matches at the start of a string, or even replace parts of the text.
  • Python‚Äôs re module helps you use regex in your code to supercharge your string manipulation skills!

Chapter 8: Reading and Writing Files

What‚Äôs a File?

  • Imagine you have a big notebook where you keep your stories, drawings, and notes. This notebook is like a file on your computer. You can open it, read from it, write in it, or even add new pages!

Opening Files with open()

  • To do anything with a file (like reading or writing), you need to open it first. Think of it like unlocking a notebook before looking inside or writing in it.
    • file = open('myfile.txt')  # Opens the file 'myfile.txt' for reading.

File Input/Output (I/O)

  • Input: Reading from a file.
  • Output: Writing to a file.

File Modes

  • When opening a file, you specify a mode to tell Python what you want to do:
    • 'r' (read mode): Opens the file for reading.
      • file = open('myfile.txt', 'r')  # Opens the file for reading.
    • 'w' (write mode): Opens the file for writing. If the file exists, it erases the content.
      • file = open('myfile.txt', 'w')  # Opens the file for writing.
    • 'a' (append mode): Opens the file for writing, but does not erase the content.
      • file = open('myfile.txt', 'a')  # Opens the file for appending.
    • 'rb' (read binary mode): Opens the file for reading binary content.
      • file = open('image.jpg', 'rb')  # Opens the file for reading binary data.
    • 'wb' (write binary mode): Opens the file for writing binary content.
      • file = open('image.jpg', 'wb')  # Opens the file for writing binary data.

Reading Files

  • There are different ways to read a file:
    • .read(): Reads the entire file.
      • file = open('myfile.txt', 'r')
content = file.read()  # Reads all the content in the file.
print(content)
    • .readlines(): Reads the file line by line, returning a list of lines.
      • file = open('myfile.txt', 'r')
lines = file.readlines()  # Reads all lines and stores them in a list.
print(lines)
    • .readline(): Reads one line at a time.
      • file = open('myfile.txt', 'r')
first_line = file.readline()  # Reads the first line only.
print(first_line)

Writing Files

  • To write to a file, you can use:
    • .write(): Writes a string to the file.
      • file = open('myfile.txt', 'w')
file.write('Hello, world!')
file.close()  # Always close the file!
    • .writelines(): Writes a list of lines to the file.
      • lines = ['Hello, world!\n', 'Python is fun!\n']
file = open('myfile.txt', 'w')
file.writelines(lines)  # Writes all the lines to the file.
file.close()

File Paths

  • A file path is the address of the file. There are two types:
    • Absolute path: The full address from the root of the file system.
    • Relative path: The path relative to the current working directory.
    file = open('foldername/myfile.txt', 'r')  # Opening a file in a folder.

Conclusion

  • Reading files is like opening a notebook to see what's written inside.
  • Writing files is like adding new things to your notebook.
  • Opening files means unlocking them first with open().
  • File modes tell the computer if you‚Äôre just looking, writing, or adding to the file.
  • File paths help you locate the file, whether it‚Äôs nearby or far away!

os and os.path Modules

  • The os module helps us talk to the operating system (like Windows or Mac), and the os.path module helps us figure out things about file paths, like if a file exists or where it‚Äôs located.
    • os.getcwd(): This tells us where we are right now in the computer‚Äôs folder world. It's like saying, "Hey, where am I right now?"

print(full_path)  # Shows 'Documents/file.txt'.

    • Renaming a File: If you want to give a file a new name, you can use os.rename(). It‚Äôs like changing the name of a box to something better.
      • os.rename('oldname.txt', 'newname.txt')  # Renames the file from 'oldname.txt' to 'newname.txt'

shutil Module ‚Äì Copying and Moving Files

  • Sometimes, we want to make a copy of a file, or we want to move it to a new folder. The shutil module is like a magic toolbox that helps us copy and move files around!
    • Copying a File: If you want to make a copy of a file, you can use shutil.copy(). It‚Äôs like having two identical toys instead of just one.

Recap of What We Learned

  • os.mkdir() helps us create new folders.

  • shutil.copy() makes copies of files, and shutil.move() helps us move files to new places.

Conclusion

  • Organizing files on a computer is just like organizing your room or your toy chest! You can walk through your folders, create new ones, delete old ones, and even copy or move things around. With Python‚Äôs os and shutil tools, you‚Äôve got everything you need to keep your files neat and tidy!

Chapter 10: Debugging

What is Debugging?

  • Imagine you're playing a game, and something isn‚Äôt working the way you want it to. Maybe your character is stuck in a wall, or something doesn‚Äôt show up. Debugging is the process of finding out what went wrong and fixing it so everything works as it should!
  • In programming, debugging helps us find and fix problems in our code‚Äîthings like when we make a mistake or something breaks. Let‚Äôs learn how we can catch and fix those errors!

Error Types

  • Sometimes, when we write code, it doesn't work because of errors. Errors are like mistakes, and there are different types of mistakes that can happen:
    • SyntaxError: This happens when we make a mistake in how we write the code, like forgetting to put a comma or a parenthesis. It's like trying to spell a word wrong and the computer says, "I don‚Äôt understand this!"
      • print("Hello world"  # Missing closing parenthesis, will cause a SyntaxError
    • IndexError: This happens when we try to access something that doesn‚Äôt exist, like trying to open a box that‚Äôs not there. For example, if we ask for the 10th item in a list that only has 5 items.
      • my_list = [1, 2, 3, 4, 5]
print(my_list[10])  # There is no 10th item, will cause an IndexError
    • TypeError: This happens when we try to do something with the wrong kind of thing. Like trying to add a number to a word (you can't add apples to oranges!).

Step 1: Reading CSV Files

import csv

In this example:

  • open('people.csv', mode='r') opens the file for reading.
  • csv.reader(file) reads the file and breaks it into rows.
  • for row in reader: goes through each row one by one and prints it.

import os

    new_name = f'photo{i + 1}.jpg'

Sending Emails Automatically:


import smtplib
from email.mime.text import MIMEText

# Email setup
sender_email = "your_email@example.com"
receiver_email = "receiver@example.com"
message = MIMEText("This is a test email.")
message["Subject"] = "Automated Email"
message["From"] = sender_email
message["To"] = receiver_email

# Sending the email
with smtplib.SMTP("smtp.gmail.com", 587) as server:
    server.starttls()
    server.login(sender_email, "your_password")
    server.sendmail(sender_email, receiver_email, message.as_string())

print("Email sent successfully!")
